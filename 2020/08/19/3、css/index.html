命名规则：使用字母，数字，或下划线和减号构成，不要以数字开头。
语法有三部分构成：选择器、属性、值 (selector {property: value})
注释 /…/

使用css样式（在html 中嵌入css的方法）
1、内联方式（行内样式）（不长使用）
在html标签中使用style属性

2、在头部引用
<style type="text/css">
/*css 语法：selector {property: value}*/
p{
    border: 2px solid blue;
    padding: 5px;
}
</style>

3、外部导入方式
<link rel="stylesheet" type="text/css" href="">

优先级：有冲突的时候，就近原则，没有冲突叠加。

（二）CSS选择器
通过选择器来控制你要控制的标签的样式。
1、html选择符（标签选择器）
就是把html标签作为选择符使用。如
p{ }网页中所有p标签都会采用这个样式。

2、class选择器
    在html标签里面增加一个属性class。然后在style里面使用.class{  }。使用

3、ID选择符
    id属性要是唯一的。在html里面加入属性id，然后使用#id{  },使用。


4、通配符选择器
*{  }

5、关联选择符（包含选择符）
    加入限定条件，进行精准改变样式。
    table a{  }

6、组合选择器
    多个标签一起使用 h1,h2,h3{  }

7、伪类选择器
对伪类进行控制。
a:link{  }
